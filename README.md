# Air Quality Data 

## Setup Environment
```bash
git clone [https://github.com/stevanonathanh/Tugas-Bangkit-Data](https://github.com/stevanonathanh/Tugas-Bangkit-Data.git)
cd Tugas-Bangkit-Data
cd Dashboard
```
## Install Packages
```bash
pip install -r requirements.txt
```
## Run Streamlit App
```bash
streamlit run dashboard.py
```
